## Format Konten per Pilar

### Tutorial (prioritas tinggi untuk YouTube + Image Search)
```
Struktur ideal:
- Intro: "X adalah..." (definisi AI-citable, 50 kata)
- Materials needed (list)
- Step-by-step dengan H3 per langkah
- Foto per step (dengan alt text deskriptif)
- FAQ section (5-7 pertanyaan)
- Produk rekomendasi + internal link
Target word count: 1.200-1.800 kata
```

### Style Guide
```
Struktur ideal:
- Intro: konteks + data tren (50 kata)
- Tabel: Occasion → Rekomendasi Hijab
- 5-7 outfit breakdown dengan foto
- Tips pemilihan warna
- Internal link ke produk & artikel terkait
Target word count: 1.000-1.500 kata
```
